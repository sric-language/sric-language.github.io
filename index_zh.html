Sric

让C++更安全、更简单

开始使用 
              import cstd::*;
              struct Point {
                  var x: Int;
                  var y: Int;
                  fun distance(t: & Point) : Float {
                      var dx = t.x - x;
                      var dy = t.y - y;
                      return sqrt(dx*dx + dy*dy);
                  }
              }

为什么选择Sric？

各项能力拉满，编程语言界的六边形战士

性能

Sric语言具有极快的运行性能，这得益于其没有垃圾收集器，支持低层内存操作的能力。在具有高效表现力的同时，依然保持低开销抽象能力。

安全

Sric语言是内存安全的，没有内存泄漏，没有悬垂指针等C++常见的内存问题。Sric的安全机制不会强制用户编写复杂且低效的代码，不用做任何事就能获得内存安全。

简单

Sric则试图尽可能减少复杂性。减少没有必要的语法特性。例如没有C++的各种版本构造函数、没有函数重载、没有模板元编程、没有多继承、没有宏等。

Sric应用

Sric有着广泛的应用领域，可轻松调用已有C/C++代码库，并被C++代码调用。可用来构建命令行软件、服务端软件、嵌入式软件、桌面软件、通过webassembly编写网页前端等。

命令行

利用Sric快速开发CLI工具。Sric可以帮助你自信地维护你的应用程序。

WebAssembly

使用Sric来替换JavaScript，网页应用也能运行的很快。

网络

可预测的性能，资源占用小。Sric非常适合网络服务开发。

嵌入式

针对低资源设备，需要低级控制而不想放弃高级便利。可以试试Sric。

Wase跨平台GUI

Wase是Sric开发的跨平台UI库，一套代码可同时编译运行在桌面端和Web浏览器中，未来将支持Android和iOS开发。Wase有着丰富的控件和优美的界面。支持配置文件创建UI，支持自定义样式。Wase使用自绘的方式现实，类似于Qt和Flutter，但是更加轻量级。Sric语言专门为其设计了逗号表达式语法，使它的声明式API类似于SwiftUI和React Native。
